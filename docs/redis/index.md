## 环境准备

### Windows

下载[Redis](https://github.com/microsoftarchive/redis/releases)

```batch
REM 启动服务
redis-server.exe redis.windows.conf
```

```batch
REM 连接服务
redis-cli.exe -h 127.0.0.1 -p 6379
```
